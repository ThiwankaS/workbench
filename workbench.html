:colorscheme gruvbox

Troubleshooting

Treesitter

:TSInstall c cpp javascript python asm
:TSUpdate

clangd diagnostics missing

Ensure compile_commands.json is in the project root.

asm_lsp not found

Install via Mason or: cargo install asm-lsp

Completion not working

Verify LSP is attached: :LspInfo. Restart with :lsp restart.
