Treesitter · Syntax

Configured in lua/treesitter.lua. AST-based highlighting and smart indent — regex syntax is disabled for code buffers so Treesitter owns colors.

ccppjavascript pythonasmlua vimbashjson

Font · JetBrains Mono

Configured in lua/font.lua — JetBrainsMono Nerd Font at 14pt for GUI Neovim (guifont).

bash ./scripts/install_jetbrains_mono_nerd.sh

In a terminal, set the same font in your emulator (Kitty, Alacritty, etc.). guifont does not apply inside TTY/SSH sessions.

Edit lua/font.lua to change size: M.size = 14

Troubleshooting

Treesitter

:TSInstall c cpp javascript python asm
:TSUpdate

clangd diagnostics missing

Ensure compile_commands.json is in the project root.

asm_lsp not found

Install via Mason or: cargo install asm-lsp

Completion not working

Verify LSP is attached: :LspInfo. Restart with :lsp restart.
