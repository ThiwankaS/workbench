Maps in lua/core/keymaps.lua, optimized for a 65% layout: home-row Space chords, no [] keys, Ctrlhjkl for windows.

Install & Setup

One-command bootstrap from the repository root:

git clone https://github.com/ThiwankaS/workbench.git ~/.config/nvim
cd ~/.config/nvim
bash ./scripts/bootstrap.sh
nvim

First launch

:checkhealth vim.pack
:Mason

Theme · Base46

68 themes via NvChad/base46. Default and options live in lua/chadrc.lua.

Spaceth
Open theme picker (Volt UI)
Spacett
Toggle between two themes in theme_toggle
-- lua/chadrc.lua
base46 = {
  theme = "gruvbox",           -- default theme name
  transparency = true,
  theme_toggle = { "gruvbox", "catppuccin" },
}

Docs inside Neovim: :h nvui · :h nvui.theme-picker

Popular Base46 themes: catppuccin, gruvbox, gruvchad, everblush, onedark, nord.

Troubleshooting

Plugins not loading

:checkhealth vim.pack
:lua vim.pack.update()

Theme looks wrong or missing

:lua require("base46").load_all_highlights()
:lua for _,v in ipairs(vim.fn.readdir(vim.g.base46_cache)) do dofile(vim.g.base46_cache..v) end

Treesitter

:TSInstall c cpp javascript python
:TSUpdate

clangd diagnostics missing

Ensure compile_commands.json is in the project root.

Completion not working

Verify LSP is attached: :LspInfo. Install servers: :MasonInstallAll.

Live grep empty

Install ripgrep: sudo apt install ripgrep
