Install & Setup

One-command bootstrap from the repo root:

bash ./scripts/bootstrap.sh

Installs system deps, copies config, runs :Lazy sync, and pre-installs Treesitter parsers for c, cpp, js/ts, markdown, and lua.

Two installers — do not confuse them

WhatCommandInstalls
Neovim plugins:Lazy / :Lazy syncLua plugins under stdpath('data')/lazy/
CLI / LSP tools:Mason → i on a packageclangd, stylua, pyright, … on your PATH via Mason

Mason “Installed (0)” means no Mason packages yet — not missing Lazy plugins.

After first launch

:Mason
:checkhealth
:Lazy clean    " optional — drop orphaned plugins from older configs

Mason auto-installs via mason-tool-installer:

clangdclang-formatlua-language-server typescript-language-serverpyrightbash-language-server marksmanstylua

C / C++ & Embedded

CMake compile database

cmake -S . -B build -DCMAKE_EXPORT_COMPILE_COMMANDS=ON
ln -sf build/compile_commands.json ./compile_commands.json

clangd warns if compile_commands.json is missing or only in build/.

ESP-IDF / FreeRTOS

  1. . $IDF_PATH/export.sh
  2. idf.py set-target esp32s3 && idf.py build
  3. Symlink compile_commands.json to project root
  4. Add project .clangd with flag Remove: list if needed
  5. In ~/.bashrc:
    export CLANGD_QUERY_DRIVER="$HOME/.espressif/tools/**/bin/*-gcc,$HOME/.espressif/tools/**/bin/*-g++"
  6. :lsp restart clangd

Assembly buffers

]m / [m next/prev label · Spaceal list labels in quickfix.

Troubleshooting

Treesitter parser missing

:TSInstall c cpp javascript typescript markdown
:TSUpdate

Needs on PATH: tree-sitter, curl, tar, C compiler.

C++ completion weak / no class members

Generate and symlink compile_commands.json (see C/C++ section).

Tab always indents / never completes

Tab is owned by blink.cmp (lua/plugins/blink.lua): menu → snippet → indent. Restart Neovim after config changes.

Alt+j / Alt+k not moving lines

Use Spacemd / mu, or fix terminal Meta key. See ttimeoutlen in lua/options.lua.

Theme / syntax colors wrong

Edit lua/config/theme.lua, then :WorkbenchThemeReload.

Old plugins still on disk

Run :Lazy clean after upgrading from a bloated config (snap, dap, snacks, etc.).

Discover keys anytime

Press Space and pause — Which-key shows available bindings.
