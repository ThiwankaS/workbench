Personal Neovim setup by ThiwankaS.

## Start Here

- Website: [https://thiwankas.github.io/workbench/](https://thiwankas.github.io/workbench/)

## Setup

```bash
bash ./scripts/bootstrap.sh
nvim
```

Then run in Neovim:

```vim
:Mason
:checkhealth
```
